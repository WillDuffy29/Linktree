# Linktree

A personal link tree built with HTML and CSS, hosting my key professional links. Built independently following the completion of my HTML course.

## Included Files

### HTML
- **index.html** — The main page. This is the 'Linktree'. 
- **placeholder.html** — A linked placeholder page for my upcoming personal portfolio website.
- **contact_me.html** — A page dedicated to emailing me. Includes a mailto: which prompts an email to my address. Easiest workaround for not utilising a database (yet).

### CSS
- **style.css** — Contains the style sheet - linked at the top of all HTML files. This was originally a basic CSS template, as I still need to proceed onto the course for it. I did, however, search up shadows and colour, their CSS syntax, and included them myself.

### Images & Media
- **profile2.jpg** — The image used for my profile picture on the site.

## Concepts Demonstrated
- Multi-page HTML structure with consistent styling.
- External CSS stylesheet linked across multiple files.
- Semantic HTML elements — header, nav, main, footer, figure.
- Anchor elements including mailto: and target="_blank".
- CSS properties including gradients, box shadows, border-radius, object-fit, and flexbox layout.
